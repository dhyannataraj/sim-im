Достаточно простенький ICQ-клиент с поддержкой многих фишек протокола v8 (ICQ 2001) для KDE или просто для QT (собирается и под виндой).

Поддерживается прием и отправка SMS, хранение списка контактов на сервере, прием и отправка сообщений в формате RTF (ICQ 2001 - передает сообщения в формате RTF), поддержка телефонной книги (в ICQ 2001 в телефонной книги можно указать доступен ли номер телефона для всех или только для "избранных"), передача файлов, чат.
В интерфейсной части особо ничего необычного нет - сделана поддержка полупрозрачных окошек (для KDE3 и Win2k), апплет для WindowMaker и поддержка "контейнеров", то есть не открывается куча окошек для каждого контакта, а несколько контактов отображаются в одном окне с закладками, в варианте для KDE используется модуль проверки орфографии.
Поддерживаются иконки от Miranda.
