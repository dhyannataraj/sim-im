SIM - Wie ändert man die Icons SIM unterstützt Icon-Themen von Miranda.
Auf der Seite http://www.nortiq.com/miranda/index.php?action=display&cat=Icon gibt es viele Themen.

Laden Sie die benötigten Dateien und entpacken Sie diese.

Bei KDE:${KDE_DIR}/share/apps/sim/icons
Bei QT:${PREFIX}/share/sim/icons
Bei MS Windows:${install_dir}/icons

Wählen Sie im Menü "Einrichten". Dann wählen Sie links "Einstellungen" und dann "Stil". Rechts können Sie dann unter "Symbole" aus einer Liste ein Icon-Thema auswählen.
