SIM - CVS Information


CVS browsen


CVS ist der Weg zum Speichern und Verfolgen der Änderungen im Quellkode zwischen den Versionen. Das CVS enthält den aktuellsten Quellkode und Bugfixes, Sie sollten aus diesem Grund das CVS verwenden. Hier ist eine schrittweise Anleitung, wie Sie SIM vom CVS herunterladen und kompilieren.

Schritt 1. Quellkode herunterladen
Führen Sie die folgenden Kommandos in einem Verzeichnis aus, in dem Sie Schreibberechtigung haben: 
cvs -d :pserver:anonymous@cvs.sim-icq.sourceforge.net:/cvsroot/sim-icq login
 (Einfach [ENTER] bei Paßwortabfrage drücken)
cvs -z3 -d :pserver:anonymous@cvs.sim-icq.sourceforge.net:/cvsroot/sim-icq checkout sim

Es werden alle heruntergeladenen Dateien angezeigt.

Schritt 2. SIM kompilieren
nachdem sie SIM vom CVS heruntergeladen haben, füren Sie die folgenden Kommandos aus: 
cd sim
gmake -f admin/Makefile.common
 Sie benötigen installiertes autoconf und automake.
Run:
./configure
gmake
gmake install

Schritt 3. SIM aktualisieren
Wenn Änderungen im Quellkode von SIM vorgenommen werden, können Sie SIM aktualisieren. Sie müssen nicht das ganze Verzeichnis löschen und neu herunterladen; Wechseln Sie in das sim-Verzeichnis und tun folgendes: 
cvs update -d
gmake
gmake install
 Dies wird die lokalen Dateien aktuelisieren und gmake wird SIM neu kompilieren.
